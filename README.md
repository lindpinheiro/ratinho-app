# Ratinho App

Projeto de exemplo (login) criado com Expo + React Native.

## Conteúdo
- Tela de login com inputs e botão de entrar
- Uso de `expo-linear-gradient`, `react-native-svg`, `@expo/vector-icons`

## Rodando localmente

Pré-requisitos: Node.js, npm, Expo CLI

1. Instale dependências

```bash
npm install
```

2. Inicie o Expo

```bash
npx expo start
```

3. Para web (opcional)

```bash
npx expo start --web --clear
```
